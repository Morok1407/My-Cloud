# SkiBin - Cloud Web Storage

This project was my graduation thesis from the Karaganda College of Innovative Technologies. My thesis title was "Development of Cloud Web Storage."

A web application for storing, managing, and accessing files through a browser.
The project provides functionality similar to cloud storage, with an emphasis on simplicity, minimal security, and extensibility.
